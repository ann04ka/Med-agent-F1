# Med-agent-F1
«Медицинский инсайт» - это AI-помощник на основе технологии LLM, предназначенный для анализа моковых медицинских данных Санкт-Петербурга и предоставления инсайтов. Участникам предлагается изучить датасет, выявить интересные закономерности и реализовать агента, который отвечает на вопросы пользователей, основываясь на данных и найденных инсайтах.
